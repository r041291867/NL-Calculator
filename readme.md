# 🍁 楓之谷標賊練功效益與配點模擬器 (MapleStory Night Lord Optimizer)

這是一個專為《楓之谷》（懷舊服/早期版本）**標賊 (Night Lord)** 設計的單網頁應用程式 (SPA)。
它可以精確模擬遊戲中複雜的傷害公式、命中率機制與多段攻擊機率，幫助玩家找出「當前裝備與等級下，打哪隻怪物練功效率最高」，並且支援即時的「配點模擬」功能。

## ✨ 核心特色

- **100% 對齊真實傷害公式**：完美還原標賊專屬的傷害系數、防禦減傷、等級懲罰以及無條件捨去（Rounddown）機制。
- **雙引擎機率計算模型**：
    - ⚡ **常態近似 (Normal Approximation)**：採用與 Excel `NORM.DIST` 完全一致的數學模型，瞬間算出擊殺機率與期望值。
    - 🎲 **蒙地卡羅 (Monte Carlo)**：內建 10,000 次暴力模擬引擎，完美還原遊戲引擎的「整數亂數」與「隨機性」，供強迫症玩家驗證絕對真實的秒殺率。
- **整合物理命中系統**：自動計算必中門檻與實際命中率。越級打怪不再是憑感覺，系統會精確告訴你 Miss 帶來的期望值損失。
- **動態配點模擬器**：內建「升/降級」按鈕與自動配點設定。你可以即時測試「點敏捷撐命中」與「全點幸運撐攻擊」在不同怪物身上的實際收益變化。
- **多段攻擊支援**：精確計算「雙飛斬（2 段）」在不同爆擊組合下的獨立傷害判定。

## 🚀 如何使用

本專案採用 **Vue 3 (CDN)** 與 **Tailwind CSS (CDN)** 開發，**完全不需安裝任何環境或依賴套件 (No Build Required)**。

1. 下載或 Clone 本專案。
2. 直接用瀏覽器（Chrome, Edge, Firefox 等）開啟 `index.html`。
3. 開始輸入你的角色素質與目標怪物資料！

_(備註：所有輸入的資料會自動使用 `localStorage` 儲存在你的瀏覽器中，重新整理不會遺失數據。)_

## 💡 參數說明與小技巧

### 角色面板

- **攻擊段數**：雙飛斬請設定為 `2`。
- **爆擊機率 / 爆擊額外**：若點滿「強力投擲」，請分別填入 `50` (%) 與 `100` (%)。
- **技能威力**：雙飛斬單下為 `150` (%)。
- **額外命中 (+ACC)**：用於填寫裝備附加、命中丸或精準之暗器提供的額外命中數值。

### 自動配點模擬

在面板右上角，你可以設定每次點擊 `+` (升級) 時，能力值要如何分配（例如：LUK: 4, DEX: 1）。

- _應用場景：當你發現打某隻怪命中率只有 70% 時，你可以試著升級並把點數投資在 DEX，觀察整體「每下經驗收益」是否比全點 LUK 還要高。_

### 怪物設定

- 對於有「多階段變身」的怪物（例如香水怪物），請將各階段拆分成不同的小怪輸入，並自行將它們算出的「期望擊殺次數」相加即可。

## 🛠️ 技術疊代筆記

本計算機的機率模型經歷了多次疊代以追求極致的準確度：

1. **均勻分佈假說**：早期版本，在處理多段攻擊時會產生尾端機率低估的誤差。
2. **二項式窮舉**：解決了雙飛斬獨立判定爆擊的問題。
3. **常態分佈近似 (CLT)**：導入平均值與變異數加總公式，完美對齊專業 Excel 計算結果。
4. **蒙地卡羅模擬**：實裝暴力窮舉引擎，補足常態近似法在小樣本 (2段攻擊) 下的微小幾何誤差。
